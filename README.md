# Pedestrian-route-choice
基于元胞自动机的单双层混合行人路径选择模型
## 模型概述

- 1.更新规则：并行更新
- 2.两种路径选择模型：
模型1：考虑路口的拥挤程度、流率大小、跟随特性。模型2：在模型1的基础上，设置行人预期选择和重新选择的规则
- 3.行人动力学模型：元胞自动机场域模型
